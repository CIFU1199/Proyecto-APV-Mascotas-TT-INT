# 🐾 Sistema de Gestión de Citas Veterinarias

_Aplicación web para agendar y administrar citas en una clínica veterinaria._

## 📌 Descripción
Sistema diseñado para:
- Agendar, modificar y cancelar citas de mascotas.


## 🛠 Tecnologías

- **Frontend**: 
  - React.ts
  - Material UI

## 📋 Requisitos
1. **Para desarrollo**:
    - Node
